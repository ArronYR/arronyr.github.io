JavaScript Window 对象

发表于 2015-01-19   |   分类于 JavaScript   |  

Window对象的方法：

  • alert()
    弹出一个警告对话框
    window.alert( message );

  • confirm()
    弹出一个选择对话框
    window.confirm( message );
    return true || false

  • prompt()
    弹出一个供用户输入信息的对话框
    window.prompt( message, defaultvalue );

  • blur()
    是window 失去焦点
    window.blur();

  • focus()
    是窗口获得焦点，调用该方法后window 窗口将成为当前窗口
    window.focus();

  • setInterval()
    指定每隔多长时间执行指定代码一次
    window.setInterval( expression, msec[arg1, arg2, …] );
    以毫秒为单位指定十佳间隔

  • clearInterval();

  • setTimeout()
    指定多长时间之后执行指定的代码
    window.setTimeout( expression, msec[arg1, arg2, …] );
    以毫秒为单位指定十佳间隔

  • clearTimeout();

  • close()
    关闭window 窗口（系统会弹出一个提示框询问是否关门窗口）
    如果想要直接官迷窗口，而不出现询问对话框，可以在 close 方法之前先设置 window 对象的 opener 属性为 null
    ( window.opener = null )
    window.close();

  • open()
    打开一个新的窗口
    window.open( [sUrl], [sName], [sFeatures] );

    1. sUrl 可选项，字符表达式，用于指定在新的窗口中显示的内容，如果没有指定测在窗口中显示的内容为空。
    2. sName 可选项，字符表达式，可以指定窗口的名称，另外该项与 HTML 的标签 具有相同的作用，可以指定窗口的打开位置和类型
      _blank 指定的内容将被显示在一个新的、没有名字的窗口中
      _media 指定的内容将被显示在 HTML 文档的媒体工具中 > IE 6.0
      _parent 指定的内容将被显示在当前框架的父框架中，如果没有父框架则该选型作用于_self 相同
      _search 指定的内容将被显示在 IE 的搜索面板中 > 5.0
      _self 指定的内容将被显示在当前窗口中
      _top 指定的内容将被取代任何将被加载的框架集，如果当前页面没有定义任何框架则该选型作用与 _self 相同
    3. sFeatures 可选项，用于指定新窗口的属性
      fullscreen 指定窗口是否全屏显示 yes || no
      height 高度
      width 宽度
      left 指定距离屏幕左边的距离
      top 指定距离屏幕上方的距离
      location 指定窗口中是否显示地址栏 yes || no
      resizable 指定窗口是否可改变大小 yes || no
      scrollbars 指定是否在窗口中显示滚动条 yes || no
      status 指定窗口中显示状态栏 yes || no
      titlebar 指定是否在窗口中显示标题栏 yes || no
      toolbar 指定在窗口中是否显示工具栏 yes || no

  • moveBy()
    通过指定偏移量来移动窗口
    window.moveBy( ix, iy );

  • moveTo()
    移动窗口到指定坐标
    window.moveTo( xp, yp );

  • navigate()
    在当前窗口中加载指定页面
    window.navigate( sUrl );

  • resizeBy()
    通过指定窗口右下角坐标的偏移量来缩放窗口
    window.resizeBy( ix, iy );

  • resizeTo()
    通过指定窗口右下角的新坐标来改变窗口的大小
    window.resizeTo( ix, iy );

  • scrollBy()
    按给定的偏移量来滚动窗口中的内容
    window.scrollBy( ix, iy );

  • scrollTo()
    滚动窗口中内容到新的位置
    window.scrollTo( ix, iy );

  • showModalDialog()
    打开一个模式对话框以显示指定内容
    window.showModalDialog( sUrl, [vArguments], [sFeatures] );

    1. sUrl 必选项，用于指定在打开的模式对话框中显示的内容
    2. vArguments 可选项，任意类型的数据，这些数据将被传递到打开的对话框中，在对话框中可以使用 dialogArguments 属性来获取传递给对话框的数据
    3. sFeatures 可选项，字符串表达式，用于指定打开的对话框的属性
      dialogHeight 整型或者浮点型，用于指定打开的模式对话框的高度
      dialogWidth 整型或者浮点型，用于指定打开的模式对话框的宽度
      dialogLeft 整型或者浮点型，用于指定打开的模式对话框的距离屏幕左边的距离
      dialogTop 整型或者浮点型，用于指定打开的模式对话框的距离屏幕上边的距离
      center 用于指定对话框是否显示在屏幕中间，默认居中
      edge 用于指定对话框的边框的类型，可用值有两个 sunken || raised 默认为 raised
      help 用于指定是否在对话框中显示与上下文相关的帮助图标，默认显示
      resizable 用于指定窗口大小是否可调，默认不可调
      scroll 用于指定是否在窗口中显示滚动条，默认显示
      status 用于指定是否在窗口中显示状态栏

  • showModallessDialog()
    打开一个非模式对话框以显示指定内容
    window.showModallessDialog( sUrl, [vArguments], [sFeatures] );
