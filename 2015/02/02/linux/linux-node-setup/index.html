Linux 下安装 NodeJS

以前都是在windows下使用，一般一个.exe 或.msi 文件就搞定了，这还是第一次自己远程Linux服务器配置NodeJS环境。首先干的第一件事就是百度，方法有很多，我最后参照了oschine和csdn上的两篇文章成功安装。下面就来说说我的安装过程。

  1. 登录服务器看了一下当前系统信息（我的是 Linux version 2.6.32-220.el6.x86_64 gcc version 4.4.6 20110731）

  2. 在窗口中输入命令下载node安装包

  3. 解压安装包

  4. 安装之前看看nodejs的所需要的环境


从上面的截图中我们可以看出NodeJS需要的环境和linux下安装方法。

  1. 因为上面下载的包是编译好的文件，里面并没有 configure这个东西，我们只需要将bin目录下的node和npm配置到全局即可

如果采用的是源码编译安装，就是使用图片中提供的安装方法。
